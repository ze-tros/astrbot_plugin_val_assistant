# 无畏契约助手 (Valorant Assistant)

AstrBot 无畏契约插件，支持 QQ/微信登录绑定、每日商店查询、商品监控、对局战绩查询和 AI 深度分析。

## 功能

- **账号绑定** — QQ / 微信扫码登录，自动同步角色信息
- **每日商店** — 查询每日商店和王国商店，生成精美图片
- **商店监控** — 自定义监控商品，上架时自动通知
- **每日推送** — 定时自动推送每日商店图片
- **战绩查询** — 查看最近 N 场对战详情（支持 @他人）
- **AI 分析** — 调用 LLM 生成深度对局分析报告
- **数据同步** — 自动同步角色、段位、赛季数据

## 命令

所有功能统一在 `/瓦` 下，空格分隔子命令：

| 命令 | 说明 |
|------|------|
| `/瓦` | 查看命令帮助 |
| `/瓦 qq` | QQ 扫码登录绑定 |
| `/瓦 wx` | 微信扫码登录绑定 |
| `/瓦 清除` | 清除已绑定的登录信息 |
| `/瓦 商店` | 查询今日商店 |
| `/瓦 商店 @某人` | 查询他人商店 |
| `/瓦 监控` | 监控功能帮助 |
| `/瓦 监控 添加 "皮肤 武器"` | 添加监控商品 |
| `/瓦 监控 删除 "皮肤 武器"` | 删除监控商品 |
| `/瓦 监控 列表` | 查看监控列表 |
| `/瓦 监控 查询` | 手动执行一次监控查询 |
| `/瓦 监控 开启` | 开启自动监控 |
| `/瓦 监控 关闭` | 关闭自动监控 |
| `/瓦 推送` | 推送功能帮助 |
| `/瓦 推送 订阅` | 在当前会话订阅每日推送 |
| `/瓦 推送 取消订阅` | 取消每日推送 |
| `/瓦 推送 状态` | 查看当前会话订阅状态 |
| `/瓦 推送 列表` | 查看全部用户订阅列表（管理员） |
| `/瓦 状态` | 查看绑定信息和赛季数据 |
| `/瓦 状态 @某人` | 查看他人绑定数据 |
| `/瓦 同步` | 同步角色和赛季数据 |
| `/瓦 战绩 [N]` | 查看最近 N 场对战 (默认10，最大50) |
| `/瓦 战绩 N @某人` | 查看他人最近 N 场对战 |
| `/瓦 分析 [N]` | AI 深度分析最近 N 场对战 (默认25，最大50) |
| `/瓦 分析 N @某人` | AI 分析他人最近 N 场 |
| `/瓦 用户` | 检测全部用户登录状态（管理员） |

## 配置

在 AstrBot 面板中配置以下参数：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `monitor_time` | 每日自动监控时间 (HH:MM) | `08:01` |
| `push_time` | 每日商店推送时间 (HH:MM) | `08:01` |
| `timezone` | 时区 | `Asia/Shanghai` |
| `bot_id` | 机器人 ID | `default` |
| `default_login_mode` | 默认登录模式 (qq/wx) | `qq` |
| `login_callback_url` | 登录回调地址 (s_url) | `http://connect.qq.com` |
| `login_u1_url` | 二维码轮询回调地址 (u1) | `http://connect.qq.com` |

## 依赖

- Python 3.12+
- aiohttp
- apscheduler
- sqlalchemy (异步)
- markdown (AI 分析报告渲染)

## 项目结构

```
main.py              # 插件入口，命令处理器
vallib/
  api/               # API 层：客户端、商店、战绩、认证、角色
  db/                # 数据层：数据库迁移、仓储
  services/          # 业务层：调度器、通知、图片生成、AI 分析
  login/             # 登录流程：QQ 扫码、微信扫码、Token 续期
  title_builder.py   # 标题构建工具
_conf_schema.json    # 插件配置 Schema
shop_template.html   # 商店图片 HTML 模板
analysis_template.html # AI 分析报告 HTML 模板
```
